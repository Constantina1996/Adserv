To be able to run API Call Assistant please follow the below intsructions:

Required tools:
1) gorgias-src-0.6d
2) prolog
3) stanford-corenlp-full-2015-12-09
4) java

Instructions:
1) Place call.pl file inside georgias A examples folder (gorgias-src-0.6d\examples)
2) Run runAPI.bat
3) Set paths to gorgias-src-0.6d, and to stanford-corenlp-full-2015-12-09
4) Enter the scenario options using available chackboxes
5) Enter your command (instructions as a user to API, e.g "decline all calls")
6) Click run
